# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Package Manager
    - **Bun**

This system is defined in modules, each module is a self-contained package that can be used independently.

## Modules

### Mobile App

Located in `mobile-app` directory.

This is the main app that will be used to interact with the system.


### Smart Contracts

Smart contracts are located in `smart-contracts` directory.

This is the system that will be used to interact with the blockchain.

### Web App

Located in `web-app` directory.

Used for: creating distributions, claiming rewards, withdrawing rewards, etc.

### Common

Common code is located in `common` directory.

This is the code that is shared between the modules.
